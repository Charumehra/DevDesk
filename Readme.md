# Dev Detective 🔎

Dev Detective is a web application that allows users to search for GitHub profiles and compare two developers in a GitHub battle. It fetches data from the GitHub API and displays useful information such as profile details, repositories, and total stars.

## 🚀 Features

* 🔍 Search any GitHub user by username
* 👤 View profile information (avatar, name, bio, join date)
* 📂 Display latest repositories
* ⚔️ GitHub Battle between two users
* ⭐ Compare users based on total repository stars
* 🏆 Highlight the winner of the battle

## 🛠️ Tech Stack

* HTML
* Tailwind CSS
* JavaScript
* GitHub REST API

## 📸 How It Works

1. Enter a GitHub username to view profile details.
2. Use the **Battle Mode** to compare two GitHub users.
3. The app calculates total stars from repositories and declares a winner.

## 📦 API Used

GitHub Public API
https://api.github.com/

## 🎯 Purpose

This project was built to practice working with APIs, asynchronous JavaScript, and dynamic UI rendering.
